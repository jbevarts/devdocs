# DevDocs AI Blog

This blog documents the development journey of DevDocs AI, showcasing the technical decisions, experiments, and learnings that went into building a production-ready AI documentation assistant.

This blog serves as:
- **Technical Documentation:** Deep dives into implementation details

- **Portfolio Showcase:** Highlights technical skills and decision-making

## Adding New Posts

1. Create a new markdown file in `posts/` directory
2. Use format: `XXX-topic-name.md` (where XXX is a 3-digit number)
3. Add entry to this README
4. Follow the template structure:
   - Title and metadata
   - Overview
   - Technical content
   - Code examples
   - Metrics/data
   - Key learnings
   - Future improvements
